關於本課程

職能發展學院 (TVDI) 的 Python 機器學習課程，專為初學者設計，從基礎 Python 語法到進階機器學習應用。

課程內容

基礎 Python

學習 Python 基本語法、資料結構與程式邏輯

  • 變數與資料型態
  • 條件判斷與迴圈
  • 函數與模組

資料處理

掌握 Pandas 與 NumPy 進行數據分析

  • 資料讀取與清理
  • 統計分析
  • 資料視覺化

機器學習

使用 Scikit-learn 建構預測模型

  • 監督式學習
  • 非監督式學習
  • 模型評估

Web 開發

用 Flask 建立互動式 Web 應用

  • 路由設計
  • 模板渲染
  • API 開發
